# 現場フォト（Genba Photo App）

工事写真台帳を「撮る → フェーズ分け → コメント → Excel出力」までスマホ1台で完結させるアプリ。

## 目的

現場監督や職人向けに、工事写真の整理と提出書類（Excel写真台帳）作成を効率化する。

## できること

- 写真の複数枚アップロード（カメラ撮影・ギャラリー・ドラッグ&ドロップ、HEIC対応）
- 施工前・施工中・施工後のフェーズ分け、グリッド/台帳ビュー切替
- 施工前の写真を画面に重ねて同じ画角で撮影できる「半透明撮影」
- 台帳ビューでのコメント入力・自動保存・テンプレート挿入
- 工事写真台帳Excel出力（通常版／施工前後比較版）
- ホーム画面に追加してアプリのように使える（PWA）

## 注意事項

写真・プロジェクト・コメントは端末内（ブラウザのストレージ）に保存されます。
**他の端末とは自動で同期されません。**（ログインのみクラウド管理）
