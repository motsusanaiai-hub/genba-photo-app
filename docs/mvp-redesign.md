# MVP再設計：コメント入力時間の最短化

作成日: 2026-05-31

---

## 設計の軸

```
旧：写真を管理して、最終的にExcelを出す
新：コメントを最速で入力して、すぐExcelを出す
```

現場シミュレーションの結果、コメント入力（40〜60分）が最大のボトルネックと判明。
ここを削ることがMVPの最優先課題。

---

## 時間短縮の構造

```
【変更前】
写真タップ → 詳細画面 → コメント欄タップ → 考える → 入力 → 保存 → 一覧に戻る
            ↑──────────────── 30回繰り返す ────────────────↑
所要時間：約40〜60分

【変更後】
台帳ビューでスクロール → コメント欄タップ → テンプレート選択 → 微修正（任意）
                        ↑──── 30回繰り返すが画面遷移ゼロ ────↑
所要時間：約10〜15分（65〜75%削減）
```

---

## 追加機能①：台帳ビュー

### 設計方針

写真一覧と全コメント入力欄を1画面に収める。詳細画面への遷移をゼロにする。

### PC・タブレット レイアウト

```
┌──────────────────────────────────────────────────┐
│ ◯◯工事  [グリッド] [台帳]  [並替]  [Excel出力]   │
├────┬─────────────────────────────────────────────┤
│ 01 │ [写真サムネイル]  施工前  2026-05-30 07:45   │
│    │ コメント: [外壁全景（補修前）          ][テ] │
├────┼─────────────────────────────────────────────┤
│ 02 │ [写真サムネイル]  施工前  2026-05-30 07:47   │
│    │ コメント: [ひび割れ状況確認            ][テ] │
├────┼─────────────────────────────────────────────┤
│ 03 │ [写真サムネイル]  施工中  2026-05-30 10:30   │
│    │ コメント: [                            ][テ] │  ← 空欄は背景色で強調
└────┴─────────────────────────────────────────────┘
```

### スマートフォン レイアウト

```
┌────────────────────────┐
│ No.01  施工前  07:45   │
│ ┌──────────────────┐   │
│ │  [写真サムネイル] │   │
│ └──────────────────┘   │
│ [コメント入力欄   ][テ]│
├────────────────────────┤
│ No.02  施工前  07:47   │
│  ...                   │
```

### 台帳ビューでできること

| 操作 | 方法 |
|---|---|
| コメント入力 | 各行の入力欄を直接編集（自動保存） |
| テンプレート選択 | [テ]ボタン → ドロップダウン |
| 上の写真と同じコメントをコピー | [テ]ボタン内に「上と同じ」オプション |
| コメント未入力の確認 | 空欄を背景色で視覚的に強調 |
| 写真の並び替え | 台帳ビュー内の▲▼ボタン |
| フェーズ確認 | 各行に施工フェーズを常時表示 |

## 追加機能②：コメントテンプレート

### テンプレートの3階層

```
レベル1：システムテンプレート（アプリが最初から用意）
  フェーズで絞り込んで表示

レベル2：カスタムテンプレート（ユーザーが登録）
  「このコメントをテンプレートに保存」で1タップ登録
  プロジェクトをまたいで使い回せる

レベル3：最近使ったコメント（自動記録）
  直近10件の使用履歴を自動表示
```

### システムテンプレート初期値

| 施工フェーズ | テンプレート |
|---|---|
| 施工前 | 全景（施工前）・補修範囲確認・既存状態確認・ひび割れ状況・劣化状況確認・墨出し確認 |
| 施工中 | 下地処理完了・清掃完了・プライマー塗布完了・材料充填完了・養生状態確認・中間検査 |
| 施工後 | 全景（施工後）・仕上がり確認・補修完了・清掃後確認・完成検査 |

### テンプレート選択UI

```
[テ]ボタンをタップ

┌─────────────────────────────┐
│ 最近使ったコメント           │
│  > 全景（補修前）            │
│  > ひび割れ状況確認          │
├─────────────────────────────┤
│ 施工前テンプレート           │  ← 現在のフェーズで絞り込み
│  > 全景（施工前）            │
│  > 補修範囲確認              │
│  > 既存状態確認              │
├─────────────────────────────┤
│ 上の写真と同じコメントを使う  │
│ カスタムテンプレートを管理    │
└─────────────────────────────┘
```

## 追加機能③：アップロード時フェーズ一括設定

```
複数写真を選択してアップロード

→「この写真の施工フェーズを選んでください」
  [施工前]  [施工中]  [施工後]
         ↓ 施工前を選択
→ 全写真に「施工前」タグが一括設定されてアップロード完了
```

1枚ずつタグを設定する手間をゼロにする。

## 改善後のシミュレーション

```
17:30 事務所到着
17:35 施工後12枚をアップロード → フェーズ「施工後」を一括設定（30秒）
17:36 台帳ビューを開く（30枚が順番に表示）
17:37〜17:50 コメント入力（約13分）
             テンプレートを選んで微修正するだけ
             空欄が視覚的に目立つので抜け漏れなし
17:50 並び順を確認・微調整（2〜3分）
17:53 Excel出力 → ダウンロード完了

作業終了：17:55（元の19:15から約80分短縮）
```
