# 工事写真管理アプリ 収益化計画

作成日: 2026-05-31
改訂日: 2026-05-31

---

## プロダクトの本質

このアプリは「工事写真アプリ」ではない。

**「工事写真台帳を楽に作るアプリ」** である。

現場の職人・監督が抱える本当の課題は「写真を撮ること」ではなく、
**「撮った写真を台帳にまとめて提出すること」** にある。

特に以下が核心機能：

- **Excel出力**（誰でも開けるファイル形式で台帳を生成）
- **写真削除時の自動行詰め・コメント連動削除**（既存ソフトにない自動処理）
- **写真自動圧縮**（重い写真ファイルを扱いやすくする）

この本質をプラン設計・価格・訴求メッセージ全体に一貫させる。

## プラン設計

### 無料プラン

**ターゲット：職人・一人親方**

収益源：広告表示

**考え方**

職人は新しいツールを使い始める前に「本当に使えるのか」を試したい。
無料プランは試用版ではなく、**それ自体で完結する便利なツール**として設計する。
写真を撮る・整理する・見返す、という日常的な使い方に困らない状態にする。

| 制限項目 | 上限 |
|---|---|
| プロジェクト数 | 5件まで |
| 写真枚数 | 合計500枚まで |
| ストレージ | 2GB |
| 広告 | あり |

**使える機能（制限なし）**

- プロジェクト作成・管理
- 写真アップロード・一覧表示
- **ガイドオーバーレイ撮影**（同画角で再撮影するための半透明ガイド）
- **施工前後比較**（スライダーで並べて確認）
- システム共通タグ付け（工種・工程・施工フェーズ・撮影場所）
- タグによるフィルタリング
- タグ一括編集

**無料に残す理由**

ガイドオーバーレイと施工前後比較は、このアプリを「試してよかった」と思わせる体験の核。
ここをケチると口コミが生まれない。建設業は現場での口コミで広がる。

### Proプラン

**月額300円（税込）**

**ターゲット：現場監督**

**考え方**

月300円は自販機2本分。会社に決裁を取らず、個人のスマホ決済で即日使い始められる価格帯。
現場監督は「便利なら自腹でも使う」層であり、**980円・1,480円のような価格帯は意思決定を遅らせる**。

「300円で全部使える」というシンプルさが武器。機能を細かく分けて出し惜しみしない。

| 制限項目 | 上限 |
|---|---|
| プロジェクト数 | 無制限 |
| 写真枚数 | 無制限 |
| ストレージ | 50GB |
| 広告 | なし |

**使える機能（無料プランの全機能＋以下）**

- **Excel出力**
  - 写真・コメント・タグを含む工事写真台帳をExcelファイルで生成
  - 写真削除時にコメントも削除・行を自動で上詰め・ページを自動で詰める
  - ページ番号の自動採番
- **PDF出力**（工事写真帳形式）
- **写真自動圧縮**（アップロード時にWebP変換・ファイルサイズを自動削減）
- **工事看板挿入**（工事名・日付・会社名を写真に合成）
- **Google Drive連携**（プロジェクトごとに自動バックアップ・同期）
- カスタムタグ作成（各カテゴリに追加可能）

**年払いオプション**

| 支払い方法 | 金額 | 月あたり |
|---|---|---|
| 月払い | ¥300/月 | ¥300 |
| 年払い | ¥2,900/年 | ¥242（約19%割引） |

### 法人プラン

**要見積（目安：月20,000円〜）**

**ターゲット：工務店・リフォーム会社・中小建設会社**

**考え方**

法人は個人の現場監督がProプランを使い始め、
「会社でも使いたい・みんなで共有したい」という需要から契約に至るケースを想定する。
いきなり法人営業はせず、**個人ユーザーからのボトムアップで法人化**を促す導線を作る。

| 制限項目 | 上限 |
|---|---|
| ユーザー数 | 5名〜（追加1名あたり月3,000円） |
| プロジェクト数 | 無制限 |
| 写真枚数 | 無制限 |
| ストレージ | 500GB〜（超過分従量課金） |
| 広告 | なし |

**使える機能（Proプランの全機能＋以下）**

- **複数ユーザーでのプロジェクト共有**
  - 閲覧者・編集者・管理者の3権限
  - プロジェクト単位でメンバーを管理
- **管理者ダッシュボード**
  - 全ユーザーの使用状況・写真枚数・プロジェクト数を一覧管理
- **会社ロゴ入り工事看板テンプレート**（独自デザインで登録）
- **Excel/PDFテンプレートの共有**（会社統一フォーマットをメンバー全員が使用）
- **請求書払い対応**（月末締め翌月払い・領収書発行）
- **優先サポート**（翌営業日以内メール回答）
- Google Drive 共有ドライブ（Shared Drive）への書き出し
- 利用データのCSVエクスポート（バックアップ・監査用）

## 事業採算性の評価

### 前提：日本の建設業市場

| 指標 | 数値 |
|---|---|
| 建設業就業者数 | 約470万人 |
| 現場監督・施工管理技士 | 約60〜80万人 |
| 中小建設会社・工務店数 | 約40万社 |
| 既存工事写真ソフトの価格帯 | 月2,000〜5,000円/人 |

### 月300円の採算ライン

| 月次売上目標 | 必要Proユーザー数 | 無料→Pro転換率3%と仮定した場合の必要無料ユーザー数 |
|---|---|---|
| ¥100,000/月（個人事業の補助収入） | 334人 | 約11,000人 |
| ¥500,000/月（一般的な個人開発の目標） | 1,667人 | 約55,000人 |
| ¥1,000,000/月（法人化の目安） | 3,334人 | 約111,000人 |

### 法人プランの採算ライン

| 契約社数 | 月額（平均¥30,000と仮定） | 月次売上 |
|---|---|---|
| 10社 | ¥30,000 | ¥300,000 |
| 30社 | ¥30,000 | ¥900,000 |
| 50社 | ¥30,000 | ¥1,500,000 |

**法人契約50社でProプラン5,000人分に相当する売上を達成できる。**

### ¥300という価格設定の評価

#### 強み

- **個人決裁できる価格帯**：月300円は会社に申請不要。スマホのサブスク感覚で即決できる
- **競合比で圧倒的な安さ**：既存ソフトの1/7〜1/17。「安すぎる理由」が逆に話題になりやすい
- **現場での口コミ速度**：建設業は現場単位で人が集まる。1人使い始めると横に広がりやすい
- **解約ハードルが低い**：安いので試しやすく、その分チャーンも少なくなりやすい

#### リスク

- **ユニットエコノミクスが薄い**：月300円ではサポートコストが収益を上回る可能性がある
- **「安い＝信用できない」リスク**：法人担当者が稟議を通す際に価格の安さがマイナスに働くことがある
- **ボリューム依存**：採算ラインに達するには数万人規模のユーザー獲得が必要
- **価格改定の難しさ**：安さで集めたユーザーは値上げに敏感。後からの値上げは解約を招く

### 採算性の総合評価

```
リスクが低い順：法人プラン > Proプラン（年払い） > Proプラン（月払い）
```

**結論：月300円単体では採算が取りにくいが、法人プランと組み合わせた構成として見れば成立する。**

具体的には以下の収益構造を目指す：

| 収益源 | 目標（12ヶ月後） | 月次売上 |
|---|---|---|
| 広告収入（無料ユーザー） | 10,000人×¥10/人 | ¥100,000 |
| Proプラン | 2,000人×¥300 | ¥600,000 |
| 法人プラン | 10社×¥30,000 | ¥300,000 |
| **合計** | | **¥1,000,000/月** |

月100万円を12ヶ月後の目標とした場合、法人10社の獲得が鍵になる。
法人10社はProユーザーの現場監督からの紹介・ボトムアップが現実的な獲得経路。

### 推奨：価格改定の出口戦略

現時点では¥300でユーザーを集め、PMF確認後に以下を検討する。

| タイミング | 対応 |
|---|---|
| 有料ユーザー500人達成時 | 現行ユーザーの料金を据え置き、新規登録を¥500に値上げ |
| 法人プラン10社達成時 | Proプランを¥500に改定（1ヶ月前に告知・年払い移行を促す） |
| 法人プラン30社達成時 | Proプランの機能を絞り、上位プランを新設することも検討 |

## グロースシナリオ

```
無料ユーザー（職人・一人親方）
  │  写真500枚に到達
  │  Excel出力したい
  │  提出書類の作成が面倒になってきた
  ▼
Proプラン ¥300/月（現場監督）
  │  「会社のみんなでも使いたい」
  │  「後輩に共有したい」
  │  「会社のフォーマットで出したい」
  ▼
法人プラン（工務店・建設会社）
```

**アップグレードのトリガーは機能制限の瞬間に差し込む。**

- 写真500枚到達 → 「Proプランで無制限に」バナーを表示
- Excel出力ボタンを押した瞬間 → アップグレード導線を表示（機能説明付き）
- 「台帳を作りたい」という行動に連動した導線設計が転換率を決める
